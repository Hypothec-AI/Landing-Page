# Hypothec AI Landing Page

## Overview

This repository contains the marketing landing page for Hypothec AI. The goal of this page is to present our value proposition, product features, and vision to early adopter clients and potential investors. It serves as a key asset in our outreach, helping us:

- Communicate our unique approach and technology
- Showcase product benefits and flexibility
- Build credibility and trust with stakeholders
- Collect interest and feedback from early users

## Purpose

The landing page is designed to:

- Attract and inform early adopter clients
- Provide a clear narrative for investor conversations
- Demonstrate our commitment to innovation and client success

## Getting Started

1. Run `npm install`
2. Run `npm run dev`
